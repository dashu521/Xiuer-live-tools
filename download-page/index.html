安装提示

  • Windows 用户：如果浏览器提示风险，请点击"保留"或"更多信息"→"仍要运行"后继续安装
  • macOS 用户：如果提示"无法打开"，请前往 系统设置 → 隐私与安全性 → 点击"仍要打开"
  • 版本选择：不确定 Mac 芯片类型？点击屏幕左上角 🍎 → 关于本机，查看"芯片"或"处理器"信息
