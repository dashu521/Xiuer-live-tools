## v1.2.1 (2026-03-11)

### 首发版本

- **基础框架整理**：完成项目架构初始化，统一项目命名
- **多平台支持**：支持抖音小店、巨量百应、视频号、小红书、淘宝直播等平台
- **自动发言功能**：预设消息模板，支持变量随机组合
- **自动弹窗功能**：智能商品弹窗，支持快捷键触发
- **自动回复功能**：关键词回复 + AI 智能回复
- **多账号管理**：支持多组账号配置，数据隔离
- **用户认证系统**：支持手机号 + 密码登录
- **礼品卡系统**：支持会员兑换码生成与兑换
- **发布流程准备**：配置 GitHub Actions 自动构建与发布
