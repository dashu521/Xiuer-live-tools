# 更新日志

## v1.5.5 (2026-03-28)

### 稳定性增强

- **短信登录契约对齐**：修复客户端短信发码、短信登录与短信重置密码请求和线上服务端契约不一致的问题
- **账号状态渲染优化**：收窄账号、直播控制、自动回复、自动弹窗等高频页面的 store 订阅范围，减少不必要渲染
- **AI 对话渲染优化**：收窄 AI 对话页消息流、输入区和消息项的状态订阅粒度

### 更新与发布

- **应用内更新重构**：统一主进程更新入口，补齐后台检查、就绪态展示与错误处理
- **发布脚本加固**：增强 Mac Release / OSS / CDN 校验链路，补齐版本一致性检查
- **高可用部署基线**：新增 auth-api 双实例 + Nginx 网关部署模板和部署自检脚本

### 工程治理

- **性能专项审计工具**：新增 auth-api 性能审计脚本、TaskManager 基准脚本和专项审计报告
- **构建前置校验**：生产构建前先校验关键环境变量，避免在构建末端失败

## v1.4.0 (2026-03-18)

### 多设备登录优化

- **单设备登录限制**：实现基于 jti 的会话级检查，同一账号在其他设备登录时自动踢下线
- **心跳检测机制**：新增心跳检测，优化踢下线通知延迟
- **旧 Token 清理**：自动清理无 jti 的旧 token，强制重新登录

### 任务管理增强

- **任务状态修复**：修复任务状态管理三大根因问题，状态显示更准确
- **stopAll 幂等性**：补齐 stopAll 幂等性，避免重复停止
- **状态映射统一**：统一自动回复状态映射与绿点一致性

### 功能修复

- **配置同步**：登录成功后立即从云端加载配置，修复多设备配置不同步问题
- **短信发送**：移除短信发送的可选参数，与阿里云控制台保持一致
- **UI 布局**：修复自动回复页面评论列表和回复预览无限延伸问题

### 代码质量

- **Lint 修复**：修复 5 个未使用变量警告
