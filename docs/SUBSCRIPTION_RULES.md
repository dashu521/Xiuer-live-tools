## 二、免费试用规则

### 2.1 试用资格

- [ ] 新注册用户自动获得试用资格
- [ ] 每个账号仅限一次试用
- [ ] 试用期间无需绑定支付方式

### 2.2 试用期限

- [ ] 标准试用期：7天
- [ ] 特殊活动期：可延长至14天
- [ ] 试用期从首次登录开始计算

### 2.3 试用功能限制

| 功能模块 | 试用期间 | 说明 |
|----------|----------|------|
| 中控台连接 | ✅ 可用 | 完整功能 |
| 自动回复 | ✅ 可用 | 完整功能 |
| 自动发言 | ✅ 可用 | 完整功能 |
| 自动弹窗 | ✅ 可用 | 完整功能 |
| 数据监控 | ⚠️ 受限 | 仅保留最近3天数据 |
| 多账号管理 | ❌ 不可用 | 仅限单账号 |
| 高级AI模型 | ❌ 不可用 | 仅限基础模型 |

### 2.4 试用到期处理

- [ ] 到期前3天发送提醒通知
- [ ] 到期后保留数据7天
- [ ] 到期后功能受限，仅可查看历史数据

## 三、会员等级与权益

### 3.1 基础会员 (Basic)

#### 定价
- [ ] 月付：¥29/月
- [ ] 年付：¥299/年（约8.5折）

#### 权益清单
- [ ] 全部基础功能无限制使用
- [ ] 数据保留30天
- [ ] 支持最多3个账号
- [ ] 标准AI模型
- [ ] 邮件支持

### 3.2 高级会员 (Premium)

#### 定价
- [ ] 月付：¥59/月
- [ ] 年付：¥599/年（约8.5折）

#### 权益清单
- [ ] 全部功能无限制使用
- [ ] 数据保留90天
- [ ] 支持无限账号
- [ ] 高级AI模型（GPT-4级别）
- [ ] 优先客服支持
- [ ] 专属功能内测资格

## 五、计费与支付规则

### 5.1 计费周期

- [ ] 月付：按自然月计费
- [ ] 年付：按年计费，到期前7天提醒续费
- [ ] 支持随时升级/降级会员等级

### 5.2 支付方式

- [ ] 支付宝
- [ ] 微信支付
- [ ] 对公转账（企业用户）

### 5.3 退款规则

- [ ] 7天内无理由退款
- [ ] 年付用户按比例退款
- [ ] 违规使用不予退款

---

## 六、账号与设备规则

### 6.1 账号共享限制

- [ ] 同一账号同一时间仅可在1台设备登录
- [ ] 新设备登录会自动踢出旧设备
- [ ] 支持设备管理查看登录历史

### 6.2 设备绑定

- [ ] 试用期间：最多绑定2台设备
- [ ] 基础会员：最多绑定5台设备
- [ ] 高级会员：最多绑定10台设备

---

## 七、特殊场景处理

### 7.1 会员升级/降级

- [ ] 升级立即生效，按剩余天数折算差价
- [ ] 降级次月生效，当前权益保持到期

### 7.2 会员暂停

- [ ] 支持最长30天暂停（年付用户）
- [ ] 暂停期间不消耗会员时长
- [ ] 暂停期间功能受限

### 7.3 企业/团队版

- [ ] 支持多账号统一管理
- [ ] 支持子账号权限分配
- [ ] 提供用量统计报表
- [ ] 联系商务获取报价

---

## 八、违规与处罚

### 8.1 违规行为

- [ ] 账号共享/倒卖
- [ ] 使用自动化工具刷量
- [ ] 违反平台使用协议

### 8.2 处罚措施

| 违规程度 | 处罚措施 |
|----------|----------|
| 轻度 | 警告 + 功能限制 |
| 中度 | 暂停会员权益7天 |
| 重度 | 永久封号，不予退款 |
