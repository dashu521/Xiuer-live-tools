## 快速开始

### 1. 下载安装

**官方下载地址**: https://download.xiuer.work/

| 平台 | 下载文件 | 说明 |
|------|----------|------|
| Windows | `Xiuer-Live-Assistant_1.3.3_win-x64.exe` | Windows 10/11 64位 |
| macOS Apple 芯片 | `Xiuer-Live-Assistant_1.3.3_macos_arm64.dmg` | M1/M2/M3/M4 Mac |
| macOS Intel | `Xiuer-Live-Assistant_1.3.3_macos_x64.dmg` | Intel 处理器 Mac |

#### 安装步骤

**Windows:**
1. 下载 `.exe` 安装包
2. 双击运行安装程序
3. 如果浏览器提示风险，点击"保留"或"更多信息"→"仍要运行"
4. 按提示完成安装

**macOS:**
1. 下载 `.dmg` 安装包
2. 双击打开磁盘映像
3. 将应用拖到"应用程序"文件夹
4. 首次打开时如果提示"无法打开"，前往 **系统设置 → 隐私与安全性 → 点击"仍要打开"**

### 2. 首次启动

1. 打开应用后，进入登录界面
2. 使用手机号注册或登录账号
3. 登录成功后进入主界面

## 账号注册与登录

### 注册账号

1. 点击登录界面的"注册账号"
2. 输入手机号码
3. 点击"获取验证码"，输入收到的短信验证码
4. 设置登录密码（6-20位，包含字母和数字）
5. 点击"注册"完成

### 登录账号

1. 输入注册的手机号
2. 输入密码
3. 点击"登录"

### 套餐说明

登录后，系统会根据您的套餐权限展示可用功能：

| 套餐类型 | 可用功能 |

| 免费版 | 基础直播控制台、单账号 |
| 专业版 | 自动发言、自动弹窗、多账号管理 |
| 旗舰版 | 全部功能（含 AI 自动回复、子账号等） |

## 直播控制台

直播控制台是核心功能，用于连接和管理各平台的直播间。

### 支持的直播平台

- 抖音小店
- 巨量百应
- 抖音团购
- 小红书
- 视频号
- 快手小店
- 淘宝直播

### 连接直播间

1. 点击左侧导航栏"直播控制台"
2. 选择要连接的平台
3. 点击"连接直播间"按钮
4. 在弹出的浏览器窗口中登录对应平台账号
5. 进入直播间中控台页面
6. 连接成功后，控制台会显示直播间状态

### 中控台功能

连接成功后，您可以：

- **查看直播状态**: 实时显示直播在线状态
- **发送消息**: 手动发送直播间消息
- **商品管理**: 查看和管理直播商品
- **评论查看**: 实时查看直播间评论
- **置顶评论**: 将重要评论置顶显示（平台支持时）

### 多账号管理

1. 点击"添加账号"可以配置多个平台账号
2. 每个账号独立运行，互不干扰
3. 可以为不同直播间设置不同的策略配置

### 断开连接

- 点击"断开连接"按钮安全关闭直播间连接
- 或直接在浏览器窗口中关闭页面

## 自动发言

自动发言功能可以预设消息模板，定时自动在直播间发送消息。

### 配置自动发言

1. 点击左侧导航栏"自动发言"
2. 选择要配置的账号
3. 点击"添加消息"创建新的发言模板

### 消息模板设置

**基础设置:**
- **消息内容**: 输入要发送的消息文本
- **发送间隔**: 设置消息发送的时间间隔（秒）
- **随机间隔**: 开启后会在设定范围内随机间隔发送

**高级变量:**
支持在消息中使用变量，实现动态内容：

| 变量 | 说明 | 示例 |
|------|------|------|
| `{time}` | 当前时间 | "现在时间是 {time}" |
| `{product}` | 当前商品名称 | "{product} 正在热卖" |
| `{price}` | 商品价格 | "只需 {price} 元" |

**随机组合:**
可以设置多条消息，系统会随机选择发送：
```
消息1: 欢迎来到直播间！
消息2: 点点关注不迷路！
消息3: 今天福利多多，不要错过！
```

### 启动自动发言

1. 确保已连接直播间
2. 勾选要启用的消息模板
3. 点击"开始"按钮
4. 系统会按照设置自动发送消息

### 暂停/停止

- 点击"暂停"临时停止发送
- 点击"停止"完全停止自动发言

## 自动弹窗

自动弹窗功能可以智能触发商品讲解弹窗，提升商品曝光率。

### 配置自动弹窗

1. 点击左侧导航栏"自动弹窗"
2. 选择要配置的账号和直播间
3. 点击"添加商品"添加要弹窗的商品

### 商品设置

**基础信息:**
- **商品名称**: 商品显示名称
- **商品ID**: 平台商品ID（可从中控台复制）
- **讲解时长**: 弹窗显示的持续时间

**触发方式:**

| 触发方式 | 说明 |
|----------|------|
| 手动触发 | 点击按钮手动触发 |
| 快捷键触发 | 设置键盘快捷键快速触发 |
| 自动轮播 | 按设定时间自动循环弹窗 |

### 快捷键设置

1. 在商品列表中点击"设置快捷键"
2. 按下要设置的快捷键组合（如 `Ctrl+1`、`Alt+Q`）
3. 点击"确定"保存

**常用快捷键示例:**
- `Ctrl+1` - 第一个商品
- `Ctrl+2` - 第二个商品
- `Alt+Q` - 主推商品

### 使用自动弹窗

**手动模式:**
- 点击商品列表中的"讲解"按钮

**快捷键模式:**
- 在直播间页面按下设置的快捷键

**自动模式:**
- 开启"自动轮播"
- 设置轮播间隔时间
- 系统会自动循环展示商品

## 自动回复

自动回复功能可以实时监听直播间评论，通过 AI 智能生成回复内容。

### 配置自动回复

1. 点击左侧导航栏"自动回复"
2. 选择要配置的账号
3. 配置 AI 服务或关键词规则

### AI 智能回复

**支持的 AI 服务:**

| 服务商 | 特点 |
|--------|------|
| DeepSeek | 国产大模型，性价比高 |
| OpenRouter | 多模型接入，灵活选择 |
| 硅基流动 | 国内服务，稳定快速 |

**配置步骤:**

1. 选择 AI 服务商
2. 输入 API Key（从对应平台获取）
3. 选择模型（如 deepseek-chat、gpt-4 等）
4. 设置回复风格（友好/专业/活泼等）
5. 点击"保存"

**回复设置:**
- **回复间隔**: 控制回复频率，避免过于频繁
- **过滤词**: 设置不回复的关键词
- **只回复提问**: 开启后只回复包含问号的评论

### 关键词回复

可以设置特定关键词的固定回复：

1. 点击"添加关键词规则"
2. 输入关键词（支持多个，用逗号分隔）
3. 输入对应的回复内容
4. 设置优先级（高优先级优先匹配）

**示例规则:**
```
关键词: 价格,多少钱,怎么卖
回复: 亲，这款商品价格很优惠，主播正在讲解，请留意哦~

关键词: 包邮,运费
回复: 全场包邮，放心下单！
```

### 启动自动回复

1. 确保直播间已连接
2. 开启"启用自动回复"开关
3. 系统会实时监听评论并自动回复

### 回复记录

- 在"回复记录"标签页查看所有自动回复历史
- 可以查看回复内容、时间和效果

## 子账号管理

子账号管理功能可以配置多个子账号，自动进入直播间进行互动。

### 添加子账号

1. 点击左侧导航栏"子账号管理"
2. 点击"添加子账号"
3. 输入子账号信息:
   - 平台类型
   - 账号名称
   - 登录凭证（Cookie/Token）
4. 点击"保存"

### 子账号分组

可以将子账号分组管理：

1. 点击"新建分组"
2. 输入分组名称
3. 将子账号拖拽到分组中

**分组用途:**
- 不同直播间使用不同分组
- 按功能分组（互动组、提问组等）
- 轮换使用，避免单一账号频繁操作

### 配置自动互动

1. 选择子账号或分组
2. 设置互动策略:
   - **进入直播间**: 自动进入指定直播间
   - **发送消息**: 定时发送预设消息
   - **点赞**: 自动点赞
   - **关注**: 自动关注主播

3. 设置互动频率和时间段
4. 点击"启动"

### 注意事项

- 子账号需要具备对应平台的正常账号权限
- 合理设置互动频率，避免被平台限制
- 建议定期更换子账号，保持活跃度

## 常见问题

### 安装问题

**Q: Windows 安装时提示风险怎么办？**

A: 这是 Windows 对未签名软件的正常提示。点击"更多信息"→"仍要运行"即可继续安装。

**Q: macOS 提示"无法打开"怎么办？**

A: 前往 **系统设置 → 隐私与安全性 → 安全性** 部分，点击"仍要打开"按钮。

### 登录问题

**Q: 收不到验证码怎么办？**

A: 请检查:
1. 手机号是否输入正确
2. 手机信号是否正常
3. 短信是否被拦截
4. 等待 60 秒后重新获取

**Q: 忘记密码怎么办？**

A: 在登录界面点击"忘记密码"，通过手机验证码重置密码。

### 连接问题

**Q: 无法连接直播间怎么办？**

A: 请检查:
1. 网络连接是否正常
2. 平台账号是否已登录
3. 账号是否有中控台权限
4. 尝试重新连接或重启应用

**Q: 连接后经常断开怎么办？**

A: 可能原因:
1. 网络不稳定
2. 平台账号在其他地方登录
3. 长时间无操作被平台断开
建议保持页面活跃，定期检查连接状态。

### 功能问题

**Q: 自动发言没有生效？**

A: 请检查:
1. 直播间是否已连接
2. 自动发言是否已启动
3. 消息模板是否已启用
4. 发送间隔设置是否合理

**Q: AI 回复不工作？**

A: 请检查:
1. AI 服务 API Key 是否正确
2. API Key 是否有足够余额
3. 网络连接是否正常
4. 查看错误日志获取详细信息

**Q: 自动弹窗没有反应？**

A: 请检查:
1. 商品 ID 是否正确
2. 是否有中控台操作权限
3. 快捷键是否与其他软件冲突

### 性能问题

**Q: 应用运行缓慢？**

A: 建议:
1. 关闭不必要的功能模块
2. 减少同时连接的直播间数量
3. 清理应用缓存
4. 确保电脑配置满足要求

**Q: 占用内存过大？**

A: 正常现象，因为应用内嵌了浏览器。建议:
1. 定期重启应用
2. 关闭不需要的直播间页面
3. 在设置中限制后台进程

### 其他问题

**Q: 如何更新到最新版本？**

A: 应用会自动检测更新。您也可以:
1. 在"设置 → 关于"中点击"检查更新"
2. 或访问官网下载最新版本

**Q: 如何反馈问题或建议？**

A: 您可以通过以下方式联系我们:
- 邮箱: support@xiuer.work
- 官网: https://xiuer.work
