# 秀儿直播助手 v1.6.0

## 📋 更新概览

本次更新重点围绕 AI 体验、自动回复知识库和多平台商品维护效率展开。

### AI 助手与体验模式

- 新增推广期 AI 体验模式，未配置个人 API Key 时也可直接体验 AI 功能
- 优化 AI 助手对话稳定性，修复连续上下文导致的异常请求问题
- 修复时尚主题下 AI 回复文字过暗、难以阅读的问题

### 自动回复与商品知识库

- 自动回复新增商品知识库链路，商品问题可优先命中知识信息再生成回复
- 新增商品知识卡结构化维护能力，支持标题、价格、优惠、卖点、别名和 FAQ
- 新增 FAQ 表格式编辑、批量导入 / 导出 / 模板复制
- 新增知识命中率、FAQ 命中率、回退原因和知识补全建议
- 增加重复话术去重 / 冷却，减少短时间内机械复读

### 商品扫描与平台支持

- 商品标题自动回填已支持：抖音小店、巨量百应、视频号、小红书千帆、小红书蒲公英、淘宝
- 商品详情扫描生成代码层能力已扩展到：抖音小店、巨量百应、视频号、小红书千帆、小红书蒲公英、淘宝
- 抖音小店 / 巨量百应自动回复默认监听来源调整为电商罗盘大屏

## 💻 系统要求

- **Windows**: 10 及以上
