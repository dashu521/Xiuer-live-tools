Windows 安装遇到安全提示怎么办？ +
由于软件未经过 Windows 应用商店签名，部分浏览器或系统可能会提示安全警告。请点击"保留"或"更多信息"→"仍要运行"后继续安装。这是正常现象，请放心使用。
macOS 提示"无法打开"怎么办？ +
请前往 系统设置 → 隐私与安全性 → 安全性 → 点击"仍要打开"。这是由于 macOS Gatekeeper 对未签名应用的安全拦截，按此步骤操作即可正常打开。
